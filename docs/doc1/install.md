# LeavesHack-Addon安装教程
## 安装MeteorClient1.21.1版本
可以在Meteor档案馆中找到旧版本的jar包
https://github.com/ManInMyVan/meteor-archive
## 下载LeavesHack-Addon
可以在Github找到最新版jar下载
https://github.com/MrBZBZ/LeavesHack/releases
## 额外下载（可选）
现在不安装这些额外模组已经不会造成崩溃
AntiAntiXray的Baritone功能需要下载Baritone才能正常显示和使用，不下载也不影响正常功能的使用  
Printer打印机功能需要下载MaLiLib(投影模组前置)和Litematica(投影模组)才能正常使用
## 安装
将所有jar文件放到MC的mods文件夹中，确保你使用的是Fabric mod加载器，启动游戏即可
