# AutoTree - 自动树场

**模块类型**: 辅助功能  
**功能简介**: 自动种植树苗并使用骨粉催熟

---

## 参数说明

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| UseDelay | 整数 (0-1000) | 使用延迟（毫秒） | 50 |
| BlocksPer | 整数 (0-4) | 每tick操作数量 | 1 |
| UseBoneMeal | 布尔 | 使用骨粉催熟 | true |
| Shape Mode | 枚举 | 渲染模式 | Both |
| Line Color | 颜色 | 边框颜色 | 白色 |
| Side Color | 颜色 | 填充颜色 | 白色 |

---

## 使用方法

1. 背包中准备树苗和骨粉（如果开启 `UseBoneMeal`）
2. 站在树场附近，开启模块
3. 模块会自动种植树苗并使用骨粉催熟
