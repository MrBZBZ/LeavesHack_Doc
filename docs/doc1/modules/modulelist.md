# ModuleList - 模块列表

**模块类型**: HUD功能  
**功能简介**: 在屏幕上显示当前启用的模块列表

---

## 参数说明

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| x | 整数 (0-1920) | X坐标 | 1910 |
| y | 整数 (0-1080) | Y坐标 | 10 |
| additional-info | 布尔 | 显示模块额外信息 | true |
| OnlyBind | 布尔 | 仅显示有绑定按键的模块 | false |
| shadow | 布尔 | 渲染文字阴影 | true |
| module-color | 颜色 | 模块名称颜色 | 白色 |
| active-color | 颜色 | 活动模块颜色 | 绿色 |
| inactive-color | 颜色 | 非活动模块颜色 | 红色 |
| background | 颜色 | 背景颜色 | 黑色半透明 |
| tag-color | 颜色 | 标签颜色 | 白色 |

---

## 使用方法

1. 开启模块后在屏幕右上角显示列表
2. 拖动调整位置（部分版本支持）
3. 开启 `OnlyBind` 可过滤只显示绑定了按键的模块
