# AutoArmorPlus - 自动穿甲

**模块类型**: 辅助功能  
**功能简介**: 自动穿戴最佳护甲，支持鞘翅自动切换

---

## 参数说明

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| Delay | 整数 (0-1000) | 操作延迟（毫秒） | 10 |
| AutoElytra | 布尔 | 自动切换鞘翅（飞行时穿鞘翅，落地穿护甲） | true |
| IgnoreBinding | 布尔 | 忽略绑定诅咒物品 | false |
| SnowBug | 布尔 | 雪地Bug模式（特定服务器使用） | false |

---

## 使用建议

1. **鞘翅切换**: 配合 `FireworkElytraFly` 使用，实现自动穿戴和脱下鞘翅
