# AutoLogin - 自动登录

**模块类型**: 辅助功能  
**功能简介**: 自动输入登录/注册密码，管理多账号

---

## 参数说明

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| AutoSave | 布尔 | 自动保存登录信息 | true |
| LoginCommand | 字符串 | 登录指令 | "l" |
| RegCommand | 字符串 | 注册指令 | "reg" |
| CpCommand | 字符串 | 改密指令 | "cp" |

---

## 使用方法

1. 首次使用时，手动输入 `/l 密码` 或 `/reg 密码 确认密码`
2. 开启 `AutoSave` 后会自动保存当前服务器的账号密码
3. 下次进入服务器时自动发送登录指令
4. 点击模块设置中的 "Accounts" 按钮可管理多账号

---

## GUI操作

- **Add**: 添加新账号
- **Remove**: 删除选中账号
- **Edit**: 修改账号信息
- 支持按服务器IP自动匹配账号
