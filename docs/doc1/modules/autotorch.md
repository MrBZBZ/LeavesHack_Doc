# AutoTorch - 自动放火把

**模块类型**: 辅助功能  
**功能简介**: 在光照不足的地方自动放置火把

---

## 参数说明

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| Delay | 整数 (0-10000) | 放置延迟（毫秒） | 50 |
| RenderRange | 整数 (0-10) | 渲染距离 | 10 |
| Range | 整数 (0-6) | 操作距离 | 5 |
| OnlyRender | 布尔 | 仅渲染不操作（预览模式） | true |
| ThroughWall | 布尔 | 穿墙放置 | false |
| CheckLightLevel | 整数 (0-15) | 光照等级阈值，低于此值放置火把 | 7 |

---

## 使用方法

1. 开启 `OnlyRender` 预览需要放置火把的位置
2. 确认无误后关闭 `OnlyRender` 开始自动放置
3. `CheckLightLevel` 设为7可在怪物生成前预防
