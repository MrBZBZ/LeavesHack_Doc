# AutoRefreshTrade - 自动刷交易

**模块类型**: 辅助功能  
**功能简介**: 自动刷新村民交易，寻找指定附魔书

---

## 参数说明

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| Range | 整数 (0-12) | 村民搜索距离 | 5 |
| WallRange | 整数 (0-12) | 穿墙搜索距离 | 5 |
| WaitMineDelay | 整数 (0-10000) | 等待挖掘延迟（毫秒） | 5000 |
| EnchantmentsList | 附魔列表 | 目标附魔列表 | 空 |
| Level | 整数 (0-5) | 目标附魔等级 | 3 |

---

## 使用方法

1. 在 `EnchantmentsList` 中添加想要的附魔（如保护、锋利等）
2. 面朝讲台（不然挖不动），打开PacketMine(数据包挖掘)
3. 设置目标等级（如保护IV设为4）
4. 确保村民是图书管理员职业
5. 开启模块后会自动打开交易界面，未找到目标附魔则破坏讲台刷新，记得多带讲台，不一定捡的到
6. 关闭PacketMine的InstantMine(秒挖)，不然讲台一放就会被挖掉
7. 找到目标附魔后自动停止（如果Level很大，会自动匹配最大值，基本上直接拉5就行）
