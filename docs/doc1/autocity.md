# AutoCity - 自动挖角

**模块类型**: 战斗功能  
**功能简介**: 自动挖掘敌人脚下的黑曜石/末地石，破坏其包围

---

## 参数说明

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| TargetRange | 整数 (0-8) | 目标搜索距离 | 6 |
| Range | 整数 (0-8) | 操作距离 | 6 |
| DoubleBreak | 布尔 | 双挖模式（同时挖两个方块） | true |
| CityDelay | 布尔 | 挖角延迟，模拟正常挖掘 | true |
| AntiCrawl | 布尔 | 自动反趴下（防止敌人趴下躲避） | true |
| PreferSelfClick | 布尔 | 优先手动点击挖掘 | true |
| Head | 布尔 | 挖掘头部位置 | false |
| Burrow | 布尔 | 挖掘黑曜石卡身方块 | true |
| Face | 布尔 | 挖掘脸部位置 | true |
| Down | 布尔 | 挖掘脚底位置 | false |
| Surround | 布尔 | 挖掘包围方块 | true |

---

## 使用方法

1. 靠近目标玩家，开启模块
2. 模块会自动寻找并挖掘目标脚下的保护方块
3. 开启 `DoubleBreak` 可以同时破坏两个方块，提高成功率
