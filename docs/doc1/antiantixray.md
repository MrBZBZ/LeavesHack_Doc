# AntiAntiXray - 雷达矿透

**模块类型**: 辅助功能  
**功能简介**: 雷达扫描矿透，绕过反矿透插件，显示周围矿物位置

---

## 参数说明

### 通用组 (General)

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| Range | 整数 (0-50) | 扫描半径（方块） | 15 |
| delay | 整数 (0-100) | 扫描间隔延迟 | 0 |
| Step | 整数 (1-2) | 遍历步长，2为跳格扫描更快 | 2 |
| baritone | 布尔 | 联动Baritone自动挖矿 | false |
| autoMine | 布尔 | 扫描完毕自动开始挖矿 | false |

### 矿物组 (Minerals)

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| Iron | 布尔 | 渲染铁矿位置 | true |
| Gold | 布尔 | 渲染金矿位置 | true |
| Diamond | 布尔 | 渲染钻石矿位置 | true |
| Lapis | 布尔 | 渲染青金石矿位置 | true |
| AncientDebris | 布尔 | 渲染远古残骸位置 | true |

---

## 使用方法

1. 开启模块后，周围矿物会在世界中以方框形式显示
2. 开启 `baritone` 后，扫描到的矿物会自动添加到Baritone的挖掘目标
3. `Step` 设为2可以加快扫描速度，但可能遗漏少量矿物
