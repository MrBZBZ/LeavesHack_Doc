# LegitNoFall - 合法防摔伤

**模块类型**: 移动功能  
**功能简介**: 自动在落地前放置水桶，防止摔落伤害

---

## 参数说明

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| checkDown | 整数 (0-3) | 检查距离（格） | 1 |
| inventorySwap | 布尔 | 背包鬼手切换水桶 | true |
| offSet | 浮点数 (0-1) | 放置偏移位移 | 0.3 |

---

## 使用方法

1. 确保背包中有水桶
2. 从高处跳下时开启模块
3. 模块会在落地前自动放置水桶
4. `checkDown` 越大，放置时机越早
