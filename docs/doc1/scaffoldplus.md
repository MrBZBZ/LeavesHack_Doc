# ScaffoldPlus - 自动搭路

**模块类型**: 移动功能  
**功能简介**: 自动在脚下放置方块，支持自适应半砖

---

## 参数说明

| 参数名 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| Rotate | 布尔 | 放置时转头 | true |
| UsingPause | 布尔 | 使用物品时暂停 | true |
| Shape Mode | 枚举 | 渲染模式 | Both |
| Line Color | 颜色 | 边框颜色 | 白色 |
| Side Color | 颜色 | 填充颜色 | 半透白 |

---

## 使用方法

1. 确保背包中有方块
2. 行走时开启模块
3. 模块会自动在脚下放置方块防止掉落
4. 支持自动识别半砖并正确放置
